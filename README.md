## ✨ Features

- ☕ Attractive homepage with modern design  
- 📖 About section for café story  
- 📋 Menu section showcasing tea & snacks  
- 🖼️ Image gallery  
- 📍 Contact section with form  
- 📱 Fully responsive layout  
- 🎨 Clean and structured UI  
- ⚡ Smooth navigation
